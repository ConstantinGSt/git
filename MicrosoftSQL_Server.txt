select - выбрать
* - все колонки
1 select * from tableName - по умолчанию все строки
2 select top 10 (strings) * from tableName     - первые 10 строк в MSQL 
* select * from tableName limit 10(strings) - первые 10 строк в PostgresSQL
3 select top 45 percent (strings) * from tableName — для других систем запрос выполняется более сложночем в  MSQL
4 select columnName, columnName,.. (or * select all column) from tableName  - выбор нужных колонок в нужном порядке
5 select top 10 (strings) columnName * from tableName - первые 10 строк в колонке  MSQL 
 *select columnName * from tableName limit 10 (strings) - первые 10 строк в колонке PostgresSQL
** через пробел от названия колонки можно написать ее псевдоним
--* вычисления со столбцами
6 select top 3 columnName columnNickName, numColumnNam*/+-num columnNickName from tableName
* в 6 примере запрашиваем первые три строки, дам псевдоним 1 колонке, производим вычисление с 
* колонкой и даем название колонке с выводом вычисления.
* select columnName as columnNickName ==  select columnName columnNickName поэтому as не используем
7 select columName+' '+ columnName  columnNickName from tableName
* +' '+ разделитель между строковыми колонками
8 select 10+15 ==  select 10+15 from tableName  запрос сработает как калькулятор
*  select '10'+'15' — 1015 MSQL отработает по правилам строк
* select '10'+'15' — MySQL интерпритирует строки как числа, если не может то заменяет строку на 0
* select '10'+'15' — PostgresSQL считает что строки складывать нельзя о_О да ну нафиг...
* при запросе строковое значение надо обрамлять одинарными кавычками в MSQL одинарными, в других базах можно и двойными
